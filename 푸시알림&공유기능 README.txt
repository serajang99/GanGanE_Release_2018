푸시알림 관련 부분
-------------------
clock활성여부가 true일때 
앱인벤터 확장 기능인 taifun notification이 푸시알림 역할을 해준다.현재는 앱을 시작하고 10초 후에 푸시알림이 뜨도록 설정이 되어있다. 푸시알림이 clock time interval에 따라서 반복되기 때문에 푸시알림이 한번 뜨면 활성여부가 false로 바뀌도록 설정을 해놨다. 활성여부는 마음대로 조정가능,,,
보통 푸시알림은 앱이 종료된 후에 뜨지만 앱인벤터에서는 실행할 수 있는 기능이 없으므로,,
푸시알림은 간간이를 돌봐주세요..!!라는 텍스트와 함께 간간이 초기화면 이미지(touch start)가 뜨게 된다.

캡쳐&공유기능
--------------
캡쳐(카메라모양)버튼을 클릭 했을 때 현재 캔버스에 나와있는 이미지가 폰에 자동으로 저장되고 공유할 수 있는 앱들이 뜬다. saveCount 전역변수가 초기에 0으로 설정되어있고 한번 캡쳐를 할때마다 1씩 늘어나서 다른이름으로 저장되게 되어있다.
notifier는 어디에 저장되는지를 보여주고 share file with message를 했을때 카카오톡 같은 경우는 file밖에 안보내지는 단점이 있다.gmail이나 메모장 같은 경우에는 message와 같이 사진이 공유된다.
